# Projeto – Avaliação de Atleta

Projeto desenvolvido no curso **DEVstart** para praticar lógica de programação e orientação a objetos em JavaScript.

## Objetivo
Criar uma aplicação que receba os dados de um atleta e calcule sua categoria, IMC e média válida, exibindo as informações no console.

## Execução
```bash
node nome-do-arquivo.js
```

## Exemplo de Saída
```
Nome: Cesar Abascal
Idade: 30
Peso: 80
Altura: 1.7
Notas: 10, 9.34, 8.42, 10, 7.88
Categoria: Adulto
IMC: 27.68166089965398
Média válida: 9.25333333
```
